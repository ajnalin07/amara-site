# Amara Website

Static storefront for Amara, designed to publish easily on free hosting.

## What is included

- A branded homepage in [index.html](/Users/nalin.ajmera/Documents/Playground/index.html)
- A full shop page in [shop.html](/Users/nalin.ajmera/Documents/Playground/shop.html)
- A working cart page in [cart.html](/Users/nalin.ajmera/Documents/Playground/cart.html)
- Brand and story content in [about.html](/Users/nalin.ajmera/Documents/Playground/about.html)
- A contact page with placeholders in [contact.html](/Users/nalin.ajmera/Documents/Playground/contact.html)
- Shared cart and product logic in [store.js](/Users/nalin.ajmera/Documents/Playground/store.js)

## Best free launch path

- Host on GitHub Pages from this repository.
- Launch this multi-page static site on GitHub Pages first.
- Use the built-in cart for storefront feel and placeholder checkout testing.
- Start real orders through inquiry links, Instagram, WhatsApp, or email.
- Add paid checkout later only after the brand and product mix are validated.

## Product updates

- Product content lives in [products.js](/Users/nalin.ajmera/Documents/Playground/products.js).
- Replace the current `name`, `price`, `description`, and `image` values there.
- Put product photos in `assets/products/`.
- Keep file names aligned with the `image` paths in `products.js`.

## Current inquiry flow

- Product inquiry buttons open an email draft to `hello@amara-placeholder.com`.
- Replace that address in [store.js](/Users/nalin.ajmera/Documents/Playground/store.js) with the real Amara email when ready.
- Replace WhatsApp, Instagram, and email placeholders on [contact.html](/Users/nalin.ajmera/Documents/Playground/contact.html).

## GitHub Pages publish

1. Push this project to a GitHub repository.
2. In GitHub, enable Pages for the repository.
3. The included workflow publishes the site automatically.

## Good first real-data pass

- Replace the imagined product prices with actual selling prices.
- Add the real Instagram, WhatsApp, and email details.
- Replace placeholder checkout with a real payment platform later.
- Update any product names you want changed from the current creative placeholders.
